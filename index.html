03 / SKILL CATEGORY
生态演进逻辑

为什么 Search Layer 会成为
SkillHub 的基础设施

Many Skills, One Shared Capability

在 SkillHub 生态中，越来越多的 Skill 需要访问本地知识。如果每个 Skill 都独立实现：

索引 / OCR / 检索 / 排序

开发成本会迅速增加，生态扩展也会变慢。

因此生态中会自然形成一层共享能力：
Content Search Layer
